v-for遍历数组

  • 遍历过程没使用索引值,只显示数组项值value in array
  • {{item}}
  • ----------------
  • 遍历过程中使用索引值(value,index) in array
  • {{index}}.{{item}}
  • ----------------

v-for遍历对象

  • 遍历对象属性的的值value in object
  • {{player}}
  • ----------------
  • 遍历对象的属性和属性值(value,key) in object
  • {{key}}.{{player}}
  • ----------------
  • 遍历对象的属性,属性值和索引值(value,key,index) in object
  • {{index}}.{{key}}.{{player}}
